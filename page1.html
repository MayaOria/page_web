Comment créer un nouveau repository (repo) dans GitHub



  1. Step 1 :

    En haut de la page d'accueil de votre compte GitHub, cliquer sur le "+" puis sur "New repository".



  2. Step 2 :

    Remplir les champs comme demandé.



    Bien vérifier la confidentialité demandée, puis cliquer sur le bouton de création.




  3. Step 3 :

    Une fenetre s'affiche, contenant le lien du repo.



    Vous pouvez maintenant utiliser votre repo, ou le partager grace au lien. Ce lien est aussi utile pour cloner le repository en local.


    4. Et voilà! Vous êtes parés!
